# Student Data Management System

This project is a simple Python application that allows a user to manage student records.

## Features
- Add new students
- View all students
- Filter students by major
- Filter students by completed course

## Data structures used
- Lists to store students
- Dictionaries to store student information
- List comprehensions for filtering by major
- Generator expressions for filtering by course

## How to run
1. Open terminal
2. Navigate to project folder
3. Run: python student_system.py
